    **is_noise function (per D-11, D-12):**
    ```python
    def is_noise(login: str, body: str) -> bool:
        if login in SKIP_BOT_LOGINS:
            return True
        if not _SUBSTANTIVE_RE.search(body):
            return True
        return False
    ```

    **Reaction extraction (per D-08, RESEARCH Pattern 4):**
    ```python
    REACTION_KEYS = ["+1", "-1", "laugh", "hooray", "confused", "heart", "rocket", "eyes"]

    def _extract_reactions(raw_reactions) -> dict[str, int]:
        if not raw_reactions:
            return {}
        return {k: raw_reactions.get(k, 0) for k in REACTION_KEYS if raw_reactions.get(k, 0) > 0}
    ```

    **GitHubExtractor class:**

    Constructor takes `repo_name: str` and optional `cache_dir: Path = DEFAULT_CACHE_DIR`. Creates `Github(auth=Auth.Token(settings.github_token))` client and calls `client.get_repo(repo_name)`.

    `extract(state: str = "all", since: Optional[datetime] = None, until: Optional[datetime] = None) -> list[Path]` method:
    1. Calls `self.repo.get_pulls(state=state, sort="updated", direction="desc")` (per D-14)
    2. Iterates lazily. For each PR:
       - If `since` and `pr.updated_at < since`: break (early-stop, per D-14)
       - If `until` and `pr.updated_at > until`: continue (skip future PRs)
       - Collect review comments via `pr.get_review_comments()` (per D-09)
       - Collect issue comments via `pr.get_issue_comments()` (per D-09)
       - For each comment: extract login (handle `comment.user is None` -> "[deleted]" per pitfall 3), filter with `is_noise()`, build CommentRecord
       - For review comments: populate file_path=comment.path, diff_hunk=comment.diff_hunk (per D-07)
       - For issue comments: file_path=None, diff_hunk=None (per D-07)
       - Extract reactions with `_extract_reactions(comment.reactions)` (per D-08)
       - Build PRRecord from PR (handle `pr.body is None` per pitfall 2)
       - Create PRFile envelope with extracted_at=datetime.now(timezone.utc)
       - Write to `self.cache_dir / f"pr-{pr.number}.json"` using `json.dumps(pr_file.model_dump(mode="json"), indent=2)` (per RESEARCH Pattern 5)
    3. Return list of written file paths

    Ensure `self.cache_dir.mkdir(parents=True, exist_ok=True)` is called before writing.

    **Do NOT extract:** review summaries or PR body as classifiable comment (per D-10).
    **Do NOT use:** `Github("token")` positional — use `Auth.Token` (per anti-pattern).
    **Do NOT call:** `list(repo.get_pulls(...))` before filtering — iterate lazily (per anti-pattern).
