```python
def _write_cache_atomic(self, cache_path: Path, pr_file: PRFile) -> None:
    tmp_name: str | None = None
    try:
        tmp_fd, tmp_name = tempfile.mkstemp(dir=cache_path.parent, suffix=".tmp")
        with os.fdopen(tmp_fd, "w", encoding="utf-8") as f:
            f.write(json.dumps(pr_file.model_dump(mode="json"), indent=2))
        os.replace(tmp_name, str(cache_path))
    except Exception:
        if tmp_name is not None:
            with contextlib.suppress(OSError):
                os.unlink(tmp_name)
        raise
```

**Constants:**
- `logger = logging.getLogger(__name__)`
- `DEFAULT_CACHE_DIR = Path(".github-pr-kb/cache")` (same as extractor)
- `DEFAULT_MODEL = "claude-3-5-haiku-latest"` (per research recommendation — cheapest, fast, sufficient)
- `VALID_CATEGORIES: set[str] = {"architecture_decision", "code_pattern", "gotcha", "domain_knowledge", "other"}` (for normalization)
- `SYSTEM_PROMPT` — a string containing the classification system prompt. Must include:
  - Instruction to respond with ONLY a JSON object, no other text, no backticks
  - The exact 5 category strings in a numbered list
  - Instruction that confidence is a float between 0.0 and 1.0
  - Instruction that summary is a single sentence
  - Example JSON format: `{"category": "gotcha", "confidence": 0.85, "summary": "One line summary"}`

**Helper function `_body_hash(body: str) -> str`:**
- Returns `hashlib.sha256(body.encode("utf-8")).hexdigest()` (per D-04)

**Helper function `_write_atomic(path: Path, data: str) -> None`:**
- Replicates the `_write_cache_atomic` pattern from extractor.py:
  - mkstemp in path.parent with ".tmp" suffix
  - Write data to fd
  - os.replace tmp to path
  - On exception: unlink tmp if it exists, re-raise

**Class `PRClassifier`:**

`__init__(self, cache_dir: Path = DEFAULT_CACHE_DIR, model: str = DEFAULT_MODEL) -> None`:
- Raises `ValueError("ANTHROPIC_API_KEY is required for classification. Set it in .env or as an environment variable.")` if `settings.anthropic_api_key` is None
- Creates `self._client = Anthropic(api_key=settings.anthropic_api_key, max_retries=2)` (per research — SDK built-in retry)
- Sets `self._cache_dir = cache_dir`
- Sets `self._model = model`
- Loads `self._index: dict[str, dict]` from `cache_dir / "classification-index.json"` if it exists, else empty dict
- Tracks counters: `self._classified_count = 0`, `self._cache_hit_count = 0`, `self._failed_count = 0`, `self._review_count = 0`

`_load_index(self) -> dict[str, dict]`:
- Reads `self._cache_dir / "classification-index.json"` if it exists
- Returns `json.loads(content)` or empty dict on FileNotFoundError
- On `JSONDecodeError`: log WARNING ("classification-index.json is corrupt, rebuilding from scratch"), return empty dict (R5 — silent index corruption risk)
- Logs at DEBUG level when loading existing index with entry count

`_save_index(self) -> None`:
- Writes `self._index` to `self._cache_dir / "classification-index.json"` using `_write_atomic`
- Uses `json.dumps(self._index, indent=2, default=str)`

`_classify_comment(self, comment: CommentRecord) -> ClassifiedComment | None`:
- If `not comment.body.strip()`: log DEBUG "skipping empty comment {comment.comment_id}", return None (Open Q — empty bodies produce useless API calls)
- Compute `h = _body_hash(comment.body)`
- If `h in self._index`: log cache hit at DEBUG, increment `self._cache_hit_count`, build ClassifiedComment from index entry with `comment_id=comment.comment_id`, `needs_review=cached_confidence < 0.75`, return it
- Truncate body for API call: `api_body = comment.body[:10_000]` (R4 — guard against extremely long comments consuming excessive tokens)
- Call `self._client.messages.create(model=self._model, max_tokens=256, system=SYSTEM_PROMPT, messages=[{"role": "user", "content": api_body}])`
- Parse response: `text = response.content[0].text`, then `result = json.loads(text)`
- On `json.JSONDecodeError`: log WARNING with raw text at DEBUG, set result to `{"category": "other", "confidence": 0.0, "summary": "classification failed"}`
- Normalize category: `category = result.get("category", "other") if result.get("category") in VALID_CATEGORIES else "other"`
- Normalize confidence: `confidence = float(result.get("confidence", 0.0))`, then `confidence = max(0.0, min(1.0, confidence))`
- Extract summary: `summary = str(result.get("summary", ""))[:200]` (truncate at 200 chars)
- Set `needs_review = confidence < 0.75` (per D-06)
- Build `ClassifiedComment(comment_id=comment.comment_id, category=category, confidence=confidence, summary=summary, classified_at=datetime.now(timezone.utc), needs_review=needs_review)`
- Update `self._index[h] = {"category": category, "confidence": confidence, "summary": summary, "classified_at": datetime.now(timezone.utc).isoformat()}` (per D-05 — only these 4 fields)
- Save index atomically after each successful classification (per research Pitfall 4)
- Increment `self._classified_count`, and `self._review_count` if needs_review
- On `anthropic.APIError` and subclasses (after SDK retries exhausted): log WARNING with error details, increment `self._failed_count`, return None (R3 — `APIError` is the base class for all SDK HTTP errors including `AuthenticationError`, `RateLimitError`, `APIStatusError`)
- Return the ClassifiedComment

`classify_pr(self, pr_number: int) -> list[ClassifiedComment]`:
- Read `self._cache_dir / f"pr-{pr_number}.json"` and parse with `PRFile.model_validate_json(content)`
- For each comment in pr_file.comments: call `_classify_comment(comment)`, collect non-None results
- Build `ClassifiedFile(pr=pr_file.pr, classifications=results, classified_at=datetime.now(timezone.utc))`
- Write to `self._cache_dir / f"classified-pr-{pr_number}.json"` using `_write_atomic` with `json.dumps(classified_file.model_dump(mode="json"), indent=2)` (per D-01)
- Return the list of ClassifiedComment

`classify_all(self) -> list[Path]`:
- Glob `self._cache_dir / "pr-*.json"` to find all cached PR files
- Extract PR numbers from filenames (parse int from `pr-N.json`)
- Initialize `output_paths: list[Path] = []`
- For each PR number: call `classify_pr(pr_number)`, append `self._cache_dir / f"classified-pr-{pr_number}.json"` to `output_paths` (R6 — explicit path collection)
- On `FileNotFoundError` for a PR: log WARNING and `continue` (skip that PR, do not append to `output_paths`)
- Call `print_summary()` (per D-07)
- Return `output_paths`

`print_summary(self) -> None`:
- Print to stdout (per D-07): "Classification complete: {classified_count} classified, {cache_hit_count} cache hits, {review_count} need review, {failed_count} failed"
- Log the same at INFO level

**Error handling:**
- `import anthropic` at top level for `anthropic.APIError` catch (base class for all SDK HTTP errors — covers `AuthenticationError`, `RateLimitError`, `APIStatusError`, etc.)
- Individual comment failures are logged and skipped (partial failure tolerance per research Open Question 3)
- Empty/whitespace-only comment bodies: skip with DEBUG log, do not count as failure
- FileNotFoundError on pr-N.json: log WARNING and skip that PR
- JSONDecodeError on classification-index.json load: log WARNING (not silent), rebuild from scratch
- Body truncation at 10,000 chars before API call to prevent context overflow on auto-generated comments
