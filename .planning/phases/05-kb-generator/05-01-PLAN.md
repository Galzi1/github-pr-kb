```python
DEFAULT_CACHE_DIR = Path(".github-pr-kb/cache")

def _write_atomic(path: Path, data: str) -> None:
    """Write data to path atomically using a temp file and os.replace."""
    tmp_name: str | None = None
    try:
        tmp_fd, tmp_name = tempfile.mkstemp(dir=path.parent, suffix=".tmp")
        with os.fdopen(tmp_fd, "w", encoding="utf-8") as f:
            f.write(data)
        os.replace(tmp_name, str(path))
    except Exception:
        if tmp_name is not None:
            with contextlib.suppress(OSError):
                os.unlink(tmp_name)
        raise
```

       a. **GenerateResult model** (Pydantic, ConfigDict(extra="ignore")):
          - written: int, skipped: int, failed: list[dict[str, str]]

       c. **_yaml_str(value: str) -> str** -- module-level helper that wraps value in double quotes, escaping internal double quotes with backslash. Used for pr_title and author in frontmatter.

       d. **KBGenerator.__init__(self, cache_dir: Path = DEFAULT_CACHE_DIR, kb_dir: Path | None = None)**:
          - Sets self._cache_dir = cache_dir
          - Sets self._kb_dir = kb_dir or Path(settings.kb_output_dir) -- import settings inside __init__ to avoid import-time errors in tests
          - Calls self._load_manifest() into self._manifest: dict[str, str]
          - Initializes self._written = 0, self._skipped = 0, self._failed: list[dict[str, str]] = []

       e. **_load_manifest(self) -> dict[str, str]** -- reads kb_dir/.manifest.json. Returns empty dict on FileNotFoundError. Logs warning and returns empty dict on JSONDecodeError.

       f. **_save_manifest(self)** -- writes self._manifest to kb_dir/.manifest.json using _write_atomic with json.dumps(indent=2).

       g. **_find_classified_files(self) -> list[Path]** -- glob("classified-pr-*.json") in cache_dir. Returns list of Paths.

       h. **_build_article(self, pr: PRRecord, comment: CommentRecord, classified: ClassifiedComment) -> str**:
          - Build YAML frontmatter with fields in this order: pr_url (pr.url), pr_title (_yaml_str(pr.title)), comment_url (comment.url), author (_yaml_str(comment.author)), date (comment.created_at.isoformat()), category (classified.category), confidence (classified.confidence), needs_review (str(classified.needs_review).lower()), comment_id (comment.comment_id).
          - After frontmatter: blank line, `# {classified.summary}` heading (D-01), blank line, comment.body (D-01).
          - If comment.diff_hunk is not None AND comment.diff_hunk is not empty string: append blank line, then fenced code block with the diff_hunk (D-03).
          - Return the full article string.

       i. **_resolve_slug(self, summary: str) -> str** -- calls slugify(summary), then checks if the slug is already used by a different comment_id in self._manifest values or exists as a file. If collision, appends -2, -3, etc. until unique (D-09). Collision check: iterate self._manifest.values() for matching category/slug.ext paths AND check filesystem.

       j. **generate_all(self) -> GenerateResult**:
          - For each classified file path from _find_classified_files():
            - Try to read and parse with ClassifiedFile.model_validate_json(). On error (OSError, ValidationError), log warning per D-18, append to self._failed with {"file": path.name, "reason": type(exc).__name__, "detail": str(exc)}, continue.
            - Load corresponding pr-{N}.json from cache_dir for the CommentRecord data. Parse as PRFile. On error, log warning and continue.
            - For each classification in classified_file.classifications:
              - Key = str(classification.comment_id) -- MUST be str for JSON key consistency (Pitfall 1)
              - If key in self._manifest: self._skipped += 1, continue
              - Find matching CommentRecord from pr_file.comments by comment_id. If not found, log warning, append to failed, continue.
              - Build article with _build_article()
              - Resolve slug with _resolve_slug()
              - category_dir = self._kb_dir / classification.category
              - category_dir.mkdir(parents=True, exist_ok=True) (D-10)
              - rel_path = f"{classification.category}/{slug}.md"
              - _write_atomic(self._kb_dir / rel_path, article)
              - self._manifest[key] = rel_path
              - self._written += 1
          - Call self._save_manifest()
          - Return GenerateResult(written=self._written, skipped=self._skipped, failed=self._failed)
